## How to deploy it for free (15 minutes)

### Recommended: GitHub Pages

You'll get a free URL like `ilianalastname.github.io` that you can share
on your resume, LinkedIn, or in recruiter conversations.

**Steps:**

1. Create a free account at github.com if you don't have one
2. Click the "+" in the top right, then "New repository"
3. Name the repository EXACTLY this: `yourusername.github.io`
   (replace "yourusername" with your actual GitHub username — this naming
   pattern is what makes the free hosting work)
4. Make it Public, then click "Create repository"
5. On the new repository page, click "uploading an existing file"
6. Drag every file from this `portfolio` folder into the upload area
   (the HTML files, CSS files — NOT this README)
7. Scroll down, click "Commit changes"
8. Wait about 2 minutes
9. Visit `https://yourusername.github.io` — your site is live

### Alternative: Netlify Drop (even faster, slightly less polished URL)

1. Go to app.netlify.com/drop
2. Drag the entire `portfolio` folder onto the page
3. You get a live URL immediately (looks like `random-words-12345.netlify.app`)
4. You can rename it to something like `iliana-portfolio.netlify.app` for free
